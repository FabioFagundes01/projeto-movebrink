# Site Institucional - MoveBrink

## 📜 Sobre o Projeto

Este é o projeto de um site institucional moderno, completo e totalmente responsivo para a **MoveBrink**, uma empresa fictícia especializada em móveis escolares e para escritório. O design foi desenvolvido com base nas referências e no feedback do cliente, focando numa apresentação profissional e clara dos produtos e da empresa.

---

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias:

- **HTML5:** Para a estrutura semântica do conteúdo.
- **CSS3:** Para a estilização, utilizando conceitos modernos como Flexbox, Grid e Variáveis CSS.
- **TypeScript:** Para adicionar interatividade, como o menu mobile e a validação de formulários.
- **AOS (Animate On Scroll):** Uma biblioteca externa para criar animações subtis durante a rolagem da página.
- **Git & GitHub:** Para controle de versão e hospedagem do código.

---

## ✨ Funcionalidades

- **Design Responsivo:** O site adapta-se perfeitamente a diferentes tamanhos de ecrã, desde telemóveis a desktops.
- **Páginas Múltiplas:** Estrutura com páginas dedicadas para Início, Produtos/Ambientes, Sobre Nós e Contato.
- **Menu Mobile:** Menu "hambúrguer" funcional para uma navegação fácil em dispositivos móveis.
- **Validação de Formulário:** O formulário de contato valida os campos do lado do cliente com TypeScript.
- **Animações de Scroll:** Elementos da página aparecem com animações suaves conforme o utilizador rola a página.
